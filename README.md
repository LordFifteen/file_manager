# Файловый менеджер на Python



Консольный файловый менеджер с изолированной рабочей директорией.



## Возможности



\- Навигация по файловой системе

\- Создание/удаление директорий

\- Создание/чтение/запись/удаление файлов

\- Копирование, перемещение, переименование файлов

\- Изолированная рабочая директория

\- Кроссплатформенность (Windows/Linux/MacOS)



## Установка и запуск



```bash

git clone https://github.com/yourusername/file\_manager.git

cd file\_manager

python main.py
